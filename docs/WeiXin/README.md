## 什么是微信小程序
小程序是一种不需要下载安装即可使用的应用，它实现了应用"触手可及"的梦想，用户扫一扫或者搜一下即可打开应用，也体现了用完即走的理念，用户不用关心安装太多应用的问题，应用将无处不在，随时可用，但又无需安装卸载
